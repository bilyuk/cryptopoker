# Roadmap

## Phase 0: Founding (Current)

- Align on product, security, and compliance baseline.
- Establish repo standards and architecture constraints.
- Define MVP acceptance criteria and launch gates.

## Phase 1: Core MVP

- Implement user auth and custody-safe wallet orchestration.
- Deliver poker table lifecycle and hand settlement.
- Add provably fair commitment/reveal primitives.
- Stand up production-grade observability and incident runbooks.

## Phase 2: Alpha Operations

- Run invite-only alpha with monitored table cohorts.
- Tune anti-fraud and collusion detection pipelines.
- Harden withdrawal and dispute workflows with manual overrides.

## Phase 3: Public Launch

- Scale table concurrency and regional infrastructure.
- Expand compliance footprint based on approved jurisdictions.
- Iterate on retention features and game formats.
