# Escrow Contracts Workspace

This workspace is the escrow-contract foundation for Base + native USDC.

## Scope in CRY-125

- Deployment config scaffold for `base-sepolia`
- ABI generation scaffold (`pnpm --filter @cryptopoker/escrow-contracts abi:generate`)
- Automated test scaffold (`pnpm --filter @cryptopoker/escrow-contracts test`)
- Placeholders for contract source under `contracts/`

Live contract logic and audited deployment scripts are implemented in follow-up issues.
